Plan de estudio

JavaScript

Práctica uno.


Clases cubiertas:
Clase 1: ¿Qué es y para qué sirve JavaScript? (Introducción a JS y su rol en desarrollo web).
Clase 2: ¿Qué es y cómo se declara una variable? (Variables con var, let, const).
Tarea: Crea un programa simple en JavaScript que declare variables para tu nombre, edad y ciudad usando let y const. Usa console.log o alert para mostrar un mensaje como: "Hola, [nombre]! Tienes [edad] años y vives en [ciudad]." Prueba reasignar una variable declarada con let y observa qué pasa si intentas reasignar una con const.
Objetivo: Familiarizarte con la declaración de variables, su uso básico y las diferencias entre let y const.

Práctica dos.


Clases cubiertas:
Clase 3: ¿Conoces todos estos tipos de datos? (Tipos como number, string, boolean, array, object).
Clase 4: Tienes que conocerlos todos (Más sobre tipos de datos y operaciones básicas).
Tarea: Escribe un programa que declare variables de diferentes tipos de datos: un número (tu edad), un string (tu nombre), un booleano (si te gusta programar), un array (tres hobbies) y un objeto (con propiedades como nombre y edad). Usa typeof para mostrar el tipo de cada variable en la consola y crea una función que convierta tu edad de string a número.
Objetivo: Comprender los tipos de datos y cómo manipularlos, además de introducirte a las funciones básicas.

Práctica tres.


Clases cubiertas:
Clase 5: ¿Cómo usar if, else en JavaScript? (Condicionales básicos).
Clase 6: ¿Qué son las condicionales en JavaScript? (Más sobre lógica condicional, posiblemente switch).
Tarea: Crea un programa que use prompt para pedirle al usuario su edad y luego use if-else para determinar si es menor de edad (menor 18), adulto joven (18-30) o adulto mayor (> 30). Muestra un mensaje personalizado según el caso con alert. Opcional: prueba agregar un switch para clasificar el día de la semana ingresado por el usuario.
Objetivo: Practicar el control de flujo con condicionales y manejar entradas del usuario.

Práctica cuatro.


Clases cubiertas:
Clase 7: ¿Cómo usar while en JavaScript? (Bucles while).
Clase 8: ¿Qué es el ciclo for en JavaScript? (Bucles for).
Tarea: Escribe un programa que use un for para imprimir los números del 1 al 10 y un while para imprimir los números pares del 0 al 20. Luego, combina ambos bucles para contar cuántos números impares hay entre 1 y 15 y muestra el resultado en la consola.
Objetivo: Dominar los bucles y entender cómo iterar para resolver problemas.

Práctica cinco.


Clases cubiertas:
Clase 9: ¿Cómo hacer un array en JavaScript? (Arrays y sus métodos básicos).
Clase 10: ¿Cómo crear una función en JavaScript? (Definición y uso de funciones).
Tarea: Crea un array con cinco nombres de amigos. Define una función que reciba el array como parámetro y use un for para imprimirlo en la consola. Luego, usa push para agregar un nuevo nombre, pop para eliminar el último y muestra el array actualizado.
Objetivo: Combinar arrays y funciones para manipular datos de manera práctica.

Práctica seis.


Clases cubiertas:
Clase 11: Aprende a crear objetos en JavaScript (Objetos y propiedades).
Clase 12: Aprende a crear clases en JavaScript (Clases y orientación a objetos).
Tarea: Define un objeto que represente un libro (título, autor, año) y una clase Libro con las mismas propiedades. Crea dos instancias de la clase y usa una función para mostrar sus detalles en la consola.
Objetivo: Entender la diferencia entre objetos literales y clases, y practicar la programación orientada a objetos.

Práctica siete.


Clases cubiertas:
Clase 20: Aprende a manipular el DOM (Introducción al DOM).
Clase 21: Lectura de nodos en JavaScript (Explorar nodos del DOM).
Tarea: Crea una página HTML simple con un título y un párrafo. Usa JavaScript para seleccionar el título con getElementById y cambiar su texto, y usa querySelector para cambiar el color del párrafo a azul. Imprime en la consola el número de nodos hijos del elemento body.
Objetivo: Introducirte a la manipulación del DOM y la interacción con elementos HTML.

Práctica ocho


Clases cubiertas:
Clase 24: 3 formas de eliminar nodos del DOM (Eliminar elementos).
Clase 25: Aprende a utilizar event listeners (Eventos en JS).
Tarea: Crea una página con una lista de tres ítems (li) dentro de un ul. Agrega un botón que, al hacer clic, elimine el último ítem de la lista usando removeChild. Usa un event listener para lograrlo.
Objetivo: Practicar la eliminación de nodos y el manejo de eventos para crear interactividad.

Proyecto Final


Para el proyecto final, te propongo construir una Lista de Tareas Interactiva, una aplicación web que integre todos los conceptos aprendidos y sea un excelente ejemplo de tus habilidades como desarrollador Front End. Aquí tienes los detalles:
Descripción: Crea una página web donde el usuario pueda agregar, marcar como completadas y eliminar tareas. Las tareas se mostrarán en una lista dinámica.
Requisitos:
HTML y CSS: Una estructura básica con un campo de entrada (input), un botón para agregar tareas y una lista (ul). Usa CSS para que sea visualmente atractiva.
Variables y Tipos de Datos: Usa un array para almacenar las tareas como objetos (cada tarea con propiedades como texto y completada).
Condicionales: Verifica si el campo de entrada está vacío antes de agregar una tarea y muestra un mensaje si lo está.
Bucles: Usa un for o while para recorrer el array y mostrar las tareas en la lista.
Funciones: Crea funciones para agregar tareas, marcarlas como completadas y eliminarlas.
DOM: Usa createElement y appendChild para añadir tareas dinámicamente al ul, y removeChild para eliminarlas.
Eventos: Agrega event listeners al botón de agregar y a cada tarea (para marcar o eliminar al hacer clic).
Pasos:
Configura el HTML con un input, un botón y un ul vacío.
Declara un array vacío para las tareas.
Define una función agregarTarea que tome el texto del input, cree un objeto tarea y lo añada al array. Luego, actualiza la lista en el DOM.
Usa addEventListener en el botón para llamar a agregarTarea al hacer clic.
Para cada tarea en el DOM, añade botones de "Completar" y "Eliminar" con sus respectivos event listeners.
Estiliza con CSS (por ejemplo, cambia el color de las tareas completadas).
Entregable: Una página web funcional que puedes abrir en el navegador y usar para gestionar tareas.
Propósito: Este proyecto te permite aplicar todo lo aprendido (variables, condicionales, bucles, funciones, DOM, eventos) en una aplicación práctica y útil. Además, es algo que podrías mostrar en un portafolio para futuros empleos como desarrollador Front End.
